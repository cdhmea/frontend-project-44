## Установка

```bash
make install
npm link
```

После установки игры становятся доступны как команды в терминале: <br>
brain-games <br>
brain-even <br>
brain-calc <br>
brain-gcd <br>
brain-progression <br>
brain-prime <br>

## Игры

| Команда             | Описание                                      |
|---------------------|-----------------------------------------------|
| `brain-games`       | Приветствие и запрос имени игрока             |
| `brain-even`        | Определение чётности числа                    |
| `brain-calc`        | Вычисление случайного выражения               |
| `brain-gcd`         | Поиск наибольшего общего делителя             |
| `brain-progression` | Поиск пропущенного числа в прогрессии         |
| `brain-prime`       | Проверка, является ли число простым           |
